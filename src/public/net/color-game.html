BBC INFOBANK

The project tests the knowledge of rgb. The user's task is to match a random rgb string with one of the six squares. Each wrong guess causes the clicked square to disappear and the loss of one point. After five rounds, the game starts again. The easy level offers the choice of three squares, the hard level has six squares to choose from.
