BBC INFOBANK

Fetches data from BBC News API. The user can select some options to customize the content they want to read.

Technologies and tools:

  • HTML and vanilla CSS(no frameworks, no flexbox or grid)
  • vanilla Java Script - Fetch API
  • Visual Studio Code

Motivation

  • practice float & clear
  • create basic webage layout without frameworks
  • learn Responsive Web Design
  • organize Java Script code in functions
  • use CSS custom properties
  • apply semantic HTML elements
